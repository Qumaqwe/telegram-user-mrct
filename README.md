# 🏪 Username Market — Telegram Mini App

Маркетплейс для покупки и продажи Telegram-юзернеймов прямо внутри Telegram.

---

## 📋 Что нужно установить (сделай это один раз)

### Шаг 1 — Установить Node.js

1. Зайди на сайт https://nodejs.org
2. Нажми на большую зелёную кнопку **"LTS"** (рекомендуемая версия)
3. Скачай и установи как обычную программу
4. Проверь установку: открой **PowerShell** (Win+R → powershell → Enter) и введи:
   ```
   node --version
   ```
   Должно написать что-то вроде `v20.x.x` — это значит всё ок

---

## 🤖 Шаг 2 — Создать Telegram-бота

1. Открой Telegram и найди **@BotFather**
2. Напиши ему `/newbot`
3. Придумай название бота (например: `Username Market`)
4. Придумай юзернейм бота (должен заканчиваться на `bot`, например: `usermarket_bot`)
5. BotFather пришлёт тебе **токен** — это длинная строка вида `7123456789:AAFxxx...`
6. **Сохрани этот токен** — он понадобится дальше

---

## ⚙️ Шаг 3 — Настроить бэкенд

1. Открой папку `backend` в проводнике
2. Найди файл `.env.example`, **скопируй его** и назови копию `.env`
3. Открой файл `.env` в блокноте и заполни:
   ```
   BOT_TOKEN=7123456789:AAFxxx...  ← твой токен от BotFather
   WEBAPP_URL=http://localhost:5173  ← пока оставь так (для тестирования)
   SECRET_KEY=любая_случайная_строка_символов_123
   PORT=3001
   ```
4. Открой **PowerShell** в папке `backend` (Shift+ПКМ → "Открыть PowerShell")
5. Введи команду для установки пакетов:
   ```
   npm install
   ```
   Подожди, пока всё скачается (1-2 минуты)

---

## 🎨 Шаг 4 — Настроить фронтенд

1. Перейди в папку `frontend`
2. Скопируй `.env.example` → `.env`
3. Открой `.env` и заполни:
   ```
   VITE_API_URL=http://localhost:3001/api
   VITE_BOT_USERNAME=usermarket_bot  ← твой юзернейм бота (без @)
   ```
4. Открой **PowerShell** в папке `frontend` и введи:
   ```
   npm install
   ```

---

## 🚀 Шаг 5 — Запустить локально (для тестирования)

Тебе нужно открыть **два** окна PowerShell:

**Окно 1 — Бэкенд** (папка `backend`):
```
npm run dev
```
Должно написать `✅ Сервер запущен на порту 3001` и `🤖 Бот запущен!`

**Окно 2 — Фронтенд** (папка `frontend`):
```
npm run dev
```
Должно написать `➜ Local: http://localhost:5173`

Открой http://localhost:5173 в браузере — увидишь интерфейс маркета!

> ⚠️ **Примечание:** Telegram требует HTTPS для мини-приложений, поэтому через браузер
> напрямую всё работает, но через Telegram нужен деплой (см. Шаг 6).

---

## 🌐 Шаг 6 — Деплой (выложить в интернет)

Для того чтобы мини-апп работал в Telegram, нужно выложить его в интернет.
**Самый простой бесплатный способ — Railway:**

### Деплой бэкенда на Railway:

1. Зайди на https://railway.app и зарегистрируйся через GitHub
2. Нажми **"New Project"** → **"Deploy from GitHub repo"**
3. Если у тебя нет GitHub — создай аккаунт на https://github.com
4. Загрузи проект на GitHub:
   - Скачай **GitHub Desktop**: https://desktop.github.com
   - Создай новый репозиторий и загрузи папку проекта
5. В Railway выбери свой репозиторий
6. Выбери папку `backend` как **Root Directory**
7. В разделе **"Variables"** добавь переменные из твоего `.env` файла
8. Нажми Deploy — через пару минут получишь URL вида `https://xxx.railway.app`

### Деплой фронтенда на Vercel:

1. Зайди на https://vercel.com и зарегистрируйся через GitHub
2. Нажми **"New Project"** → выбери свой репозиторий
3. Укажи **Root Directory: frontend**
4. В **Environment Variables** добавь:
   - `VITE_API_URL` = `https://твой-backend.railway.app/api`
   - `VITE_BOT_USERNAME` = `твой_юзернейм_бота`
5. Нажми Deploy — получишь URL вида `https://xxx.vercel.app`

---

## 🔗 Шаг 7 — Подключить мини-апп к боту

1. Открой @BotFather в Telegram
2. Напиши `/mybots` → выбери своего бота
3. Нажми **"Bot Settings"** → **"Menu Button"** → **"Configure menu button"**
4. Введи URL твоего фронтенда (от Vercel): `https://xxx.vercel.app`
5. Введи название кнопки: `Открыть маркет`

Также обнови `.env` в бэкенде на Railway:
```
WEBAPP_URL=https://xxx.vercel.app
```

---

## 🎉 Готово!

Теперь напиши своему боту `/start` — появится кнопка **"Открыть маркет"**.
Нажми её — откроется твой маркетплейс юзернеймов!

## ❓ Частые вопросы

**Q: Не запускается npm install — ошибка**
A: Убедись что Node.js установлен (`node --version`). Если ошибка о правах — запусти PowerShell от имени администратора.

**Q: Бот не отвечает**
A: Проверь правильность токена в `.env`. Бот может работать только в одном месте одновременно.

**Q: Как реально передать юзернейм покупателю?**
A: Инструкция для продавца: Telegram → Настройки → Редактировать профиль → Юзернейм → удали старый юзернейм. Покупатель быстро его вводит себе. Это стандартный способ, как делают все.

**Q: Что такое Telegram Stars?**
A: Это внутренняя валюта Telegram. 1 Star ≈ $0.013. Пользователи покупают их в приложении через App Store / Google Play.
